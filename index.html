El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

CSS: ¿Qué es, Para qué sirve y Cómo aplicarlo?

La hoja de estilo en cascada o CSS es un lenguaje usado para definir la presentación de un documento estructurado escrito en HTML y derivados.

Un editor HTML es un software para editar y crear código HTML que se utiliza para sitios web u otros documentos web. Con los editores HTML basados en texto, el código fuente puede ser editado directamente.
